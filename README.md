# Therese Hommedal - Makeup Artist Website
_Utviklet av ehlma consulting SB_

Dette er en pågående utviklingsversjon av nettsiden for makeupartist, Therese Hommedal, designet og utviklet for å vise frem portefølje, tjenester og kontaktinformasjon på en elegang og brukervennlig måte. 

Nettsiden bygges med fokus på visuell balanse, god struktur og moderne frontend-teknologi. 

## Status
Prosjektet er under aktiv utvikling. 
Frontend er hovedfokus, backend for kontaktskjema og e-posthåndtering testes lokalt. 

## Teknologier
- Vite + React
- Tailwind CSS + modulbaserte .css-filer per seksjon for å bryte opp i lengden av css
- Cloudflare Turnstile (anti-span/honeypot på kontakt)
- Node + Express i backend

## Videre planer
- Optimalisere bilder til .webp / .avif
- Fullføre styling og responsivitet
- Fullføre e-post tjeneste
